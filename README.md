# AI Resume Reviewer

A Smart AI Resume Reviewer where you can paste Job Requirement, Upload Resume, and let AI instantly evaluate and match candidates to the right roles.

## Tech Stack

- React JS
- React Router
- Tailwind CSS
- Puter JS
